Angular环境配置和脚手架构建

1、搭建环境

  • Node.js --- Angular 需要 Node.js 相对应的版本(详见Angular官方文档)
  • npm 包管理器 --- Angular、Angular CLI 和 Angular 应用都依赖于 npm 包中提供的特性和功能。要想下载并安装 npm 包，你必须拥有一个 npm 包管理器。

2、脚手架构建

  • npm install -g @angular/cli --- 全局安装 Angular CLI
  • npm install -g @angular/cli --- 全局安装 Angular CLI
